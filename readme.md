# PIM (Product Information Management)

## Stack technologiczny

- Docker i Docker Compose
- PHP 8.3
- PostgreSQL 16
- Symfony 7.4.7

---

## Uruchamianie projektu

`cd pim`

`docker compose up`

## Parametry kontenera

- host: db (wewnątrz dockera) / localhost
- port: 5432
- user bazodanowy: user
- hasło: password
- nazwa bazy danych: app
- URL: http://localhost:8080/
- Swagger (API Platform): http://localhost:8080/api
